# DJ KARIKA Party Planning Website

موقع رسمي responsive لـ **DJ KARIKA Party Planning**. مناسب للموبايل والتابلت والكمبيوتر.

## يشمل
- Showreel في الرئيسية يعمل تلقائيًا بدون صوت وفق سياسات المتصفح.
- جاليري 8 فيديوهات YouTube؛ كل فيديو له كارت مستقل ويفتح داخل مشغل أنيق.
- قسم الخدمات: تنظيم شامل، DJ وصوت، لايت/ليزر/شاشات، استيدج وديكور، تصوير جوي وأرضي، وفاير شو.
- نموذج حجز يحوّل تفاصيل الطلب إلى WhatsApp على `01114880069`.
- روابط Facebook وTikTok وInstagram.

## تشغيل محلي
افتح `index.html` مباشرة، أو استخدم إضافة Live Server داخل VS Code.

## تعديل الفيديوهات
من `index.html` ابحث عن `data-video="..."` واستبدل فقط الـ YouTube Video ID. صورة المعاينة تستخدم نفس الـ ID ضمن الرابط `i.ytimg.com/vi/...`.

## GitHub Pages
من إعدادات الريبو: **Settings → Pages → Deploy from a branch → main → /(root) → Save**.
بعد دقائق يصبح الموقع متاحًا من رابط GitHub Pages الذي يظهر في نفس الصفحة.
